============
Penaltymodel
============

This package was deprecated in release 1.3.0 (Ocean 9.0) and will be removed in
Ocean 10.

For solving problems with constraints, we recommend using the hybrid solvers in
the Leap service. You can find documentation for the hybrid solvers at
:ref:`opt_index_hybrid`.

The ``penaltymodel`` package is
`here <https://github.com/dwavesystems/penaltymodel>`_. It is no longer
maintained.
