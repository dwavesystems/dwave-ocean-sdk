==============
dwave-networkx
==============

This package was deprecated in release 0.8.19 (Ocean 9.4) and will be removed in
Ocean 10.

``dwave-networkx`` functionality is now available in the :ref:`index_graphs`
package.

The `dwave-networkx` source code is
`here <https://github.com/dwavesystems/dwave-graphs/tree/5f45f2d68a56f9f94412fbc0e25eeb7f260b3077/dwave_networkx>`_.
It is no longer maintained.
